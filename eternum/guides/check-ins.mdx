description: "Keep your account active and understand escalation if check-ins are missed."

---

Check-ins tell Eternum that you are active. If you miss check-ins, the death verification pipeline escalates in stages.

## Check-in intervals by plan

| Plan | Check-in interval |
| --- | --- |
| Seed | 30 days |
| Legacy | 7 days |
| Estate | 3 days |

## Escalation stages

| Level | What happens |
| --- | --- |
| L0 | You receive a check-in reminder. |
| L1 | Trusted contacts receive a welfare-check prompt. |
| L2 | Trusted contacts can access verification tools. |
| L3 | Signal threshold can move the case toward challenge. |
| L4 | A challenge window opens before verified recovery. |
| Verified | Recovery shares can be released to beneficiaries. |

<Info>
A successful check-in cancels an active pending verification pipeline and returns the account to active status.
</Info>

## Medical holds

Medical holds pause normal escalation temporarily. They are intended for known events such as hospitalization or travel where check-ins may be missed.
