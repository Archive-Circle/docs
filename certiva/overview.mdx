## What is Certiva?

Certiva is an institutional credential registry and verification system. Institutions issue academic or professional credentials into a tamper-evident registry. Each credential carries structured metadata, a verification identifier, a QR payload, and a public verification URL.

Relying parties — employers, other institutions, or verification services — can confirm credential authenticity without contacting the issuing institution directly.

## Architecture

Certiva is a pnpm monorepo orchestrated with Turborepo:

| App / Package | Role |
|---|---|
| `apps/api` | NestJS 11 backend — auth, RBAC, credential registry, verification, audit |
| `apps/web` | Next.js 16 operator dashboard and public verification UI |
| `apps/worker` | BullMQ background processor — batch issuance, blockchain anchoring |
| `packages/contracts` | Solidity smart contract and deployment artifacts |
| `packages/types` | Shared TypeScript domain types |
| `packages/ui` | Shared React component library |
